Este trabalho foi feito para praticar Java no IntelliJ IDEA e para testar o uso de uma impressora por meio de uma DLL que faz a ligação entre o programa e o equipamento. 
O sistema funciona pelo próprio terminal: aparece um menu, e cada número representa uma ação diferente. Dá pra iniciar a comunicação com a impressora, mandar textos, gerar QR Code, 
imprimir código de barras, usar arquivos XML do SAT, avançar o papel, abrir a gaveta e pedir o corte no fim.
Para rodar, basta abrir o projeto, deixar a DLL no lugar correto do computador e executar a classe principal. 
Depois disso, é só seguir as opções que aparecem na tela. 
A impressora precisa estar conectada e reconhecida pelo sistema para que tudo funcione sem problema.
